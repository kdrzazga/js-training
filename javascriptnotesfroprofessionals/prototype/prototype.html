In the conventional JS there are no classes, instead we have prototypes. Like a class, prototype inherits the
properties including methods and variables declared in the class. We can create the new instance of the
object whenever it is necessary using Object.create(PrototypeName); (we can give the value for the constructor as
well)
It's similar to Prototype pattern.
